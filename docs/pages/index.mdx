# 🧠 AgentVectorDB

> The Cognitive Core for Your AI Agents

AgentVectorDB is a lightweight, embeddable vector database designed specifically for AI agents. It provides efficient memory storage, retrieval, and context management built on LanceDB.

## 🌟 Key Features

### 💫 Purpose-Built for Agents
- Optimized for agent memory and context management
- Flexible schema design for various agent types
- Built-in importance scoring and metadata

### ⚡️ High Performance
- Fast vector similarity search
- Efficient memory management
- Minimal resource footprint

### 🔄 Async First
- Native async/await support
- Perfect for real-time agent interactions
- Efficient concurrent operations

### 🛠 Developer Friendly
- Simple, intuitive API
- Extensive documentation
- Ready-to-use examples

## 🚀 Quick Start

```python
from agentvectordb import AgentVectorDBStore

# Initialize store
store = AgentVectorDBStore(db_path="./agent_memory")

# Create collection
collection = store.get_or_create_collection("thoughts")

# Add memory
collection.add(
    content="Planning next steps",
    type="thought",
    importance_score=0.9
)
```
