## Prerequisites

Before installing AgentVectorDB, ensure you have:

- Python 3.8 or higher installed
- pip (Python package installer)
- Basic understanding of vector databases and embeddings

> [!tip]
> Not sure about your Python version? Run `python --version` in your terminal.

## Installation

### 1. Install Python (if needed)

#### On macOS
```bash
brew install python@3.11
```

#### On Ubuntu/Debian

```bash
sudo apt update
sudo apt install python3.11 python3-pip
```

#### On Windows

Download Python from python.org or use:

```bash
winget install Python.Python.3.11
```


##  Create a Virtual Environment


```bash

# Create a new directory for your project
mkdir my-agent-project
cd my-agent-project

# Create a virtual environment
python -m venv venv

# Activate the virtual environment
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate

```
## Install AgentVectorDB

```bash 
pip install agentvectordb
```

##  Quick Example
Here's a minimal example to get you started:

```python filename="quickstart.py"
from agentvectordb import AgentVectorDBStore

# Initialize store
store = AgentVectorDBStore(db_path="./my_db")
```
##  Async Support
AgentVectorDB also provides async APIs for use with async frameworks:

```python
import asyncio
from agentvectordb import AsyncAgentVectorDBStore

async def main():
    store = AsyncAgentVectorDBStore(db_path="./my_async_db")
    collection = await store.get_or_create_collection("async_memories")
    
    # Add memory asynchronously
    await collection.add(
        content="Processing data asynchronously",
        type="system_log"
    )

    # Query asynchronously
    results = await collection.query(
        query_text="async processing",
        k=1
    )
    
    print(results)

if __name__ == "__main__":
    asyncio.run(main())
```
