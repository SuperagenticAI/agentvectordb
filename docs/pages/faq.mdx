## What is AgentVectorDB?

A lightweight, embeddable vector database for agentic AI systems, built on LanceDB.

## What Python versions are supported?

Python 3.8 and above.

## Can I use my own embedding model?

Yes! You can plug in any embedding function compatible with your agent framework.

## Where can I get help?

Open an issue on [GitHub](https://github.com/superagenticai/agentvectordb) or join our community!
